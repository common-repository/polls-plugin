== Installation ==

Install the plugin and get it work in less than a minute.

1. Unzip the downloaded archive from Wordpress Directory.
2. Extract **polls-plugin.zip** and upload the folder to */wp-content/plugins/*
3. Activate the Polls Plugin from *Plugins > Polls Plugin > Activate*, under WordPress admin interface
4. After activating visit the new Polls page that has been created in the interface of your WP Menu and create a poll.
